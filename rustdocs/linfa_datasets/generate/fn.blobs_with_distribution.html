Expand description

Given an input matrix blob_centroids, with shape (n_blobs, n_features), generate blob_size data points (a “blob”) around each of the blob centroids.

More specifically, each blob is formed by blob_size points sampled from a distribution centered in the blob centroid.

blobs can be used to quickly assemble a synthetic dataset to test or benchmark various clustering algorithms on a best-case scenario input.
